# ClassA

Usage of classA is as follow:

```js
import ClassA from 'demo/src/ModuleA/PackageA/ClassA';

const hello = new ClassA();
hello('seb');
```

## hello(name = 'world')

The method will say hello using the provided name.

```js
const hello = new ClassA();
hello();
=> hello word
hello('seb');
=> hello seb
```
